CSS interview Q&A

What is CSS? Why do we use it?

CSS(Cascading Style Sheets) is the language used to style an HTML document.
It is used to describe the visual presentation of a document.

In how many ways can we create css styling?What are they?

There are 3 ways to insert css into html.
● Inline styling
● Internal styling
● External styling

what is the difference b/w 3 ways of css styling?

Inline Styling: In inline styling CSS styles directly apply to the individual HTML elements using the style attribute.

Internal Styling: Internal styling in CSS refers to the method of embedding CSS rules directly within the HTML document using the style tag.

External Styling: External CSS is used to style multiple HTML pages with a single style sheet. External CSS contains a separate CSS file with a .css extension.

What is specificity among the ways of css styling?

inline > internal > external

What is the syntax of css?

selector{property:value;}
ex: h1{color:green;}

What is a selector?

A CSS selector selects the HTML element(s) you want to style.

How many types of selectors?

We can divide CSS selectors into five categories:
● Simple selectors (select elements based on name, id, class)
● Combinator selectors (select elements based on a specific relationship between them)
● Pseudo-class selectors (select elements based on a certain state)
● Pseudo-elements selectors (select and style a part of an element)
● Attribute selectors (select elements based on an attribute or attribute value)

What are the general selectors?

General selectors also known as simple selectors.
1. Element Selectors (p/h1)
2. Universal Selectors (*)
3. Id selectors (#)
4. class selectors (.)
5. grouping selectors (h1,h2)

What is difference b/w the pseudo class vs pseudo element?

Pseudo-Class
Definition: A pseudo-class is used to define a special state of an element.
Syntax: It is written with a single colon (:) followed by the pseudo-class name.
Examples:
:hover , :focus

Pseudo-Element
Definition: A pseudo-element is used to style a specific part of an element.
Syntax: It is written with a double colon (::) followed by the pseudo-element name
Examples:
::before , ::after

What is the rule set?

A CSS rule set contains one or more selectors and one or more declarations.

What is box-modal?

The CSS box model is a box that wraps around every HTML element. It consists of content, padding, borders and margins.

What are the box-modal properties? (explain)

Content: This is the actual content of the element, such as text, images, or other media.
Padding: Padding is the space between the content and the element's border. It adds internal space within the element.
Border: The border surrounds the padding and content of the element.
Margin: Margins are the space outside the element, separating it from other elements.

CSS gradients let you display smooth transitions between two or more specified colors.

Linear Gradients (goes down/up/left/right/diagonally)

float - It positions an element to the left or right and allows text or other inline elements to wrap around it.

what is d/f b/w the visibility:hidden and display:none?

visibility: hidden: The element is hidden but still takes up space in the layout. Other elements won't move.
display: none: The element is hidden and removed entirely from the document flow, meaning its space is no longer occupied and the surrounding elements adjust accordingly.

what is object-fit ? and in how many types can we fit the object?

The CSS object-fit property is used to specify how an img or video should be resized to fit its container.
fill - This is default. The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit
contain - The image keeps its aspect ratio, but is resized to fit within the given dimension
cover - The image keeps its aspect ratio and fills the given dimension. The image will be clipped to fit
none - The image is not resized
scale-down - the image is scaled down to the smallest version of none or contain

What is background property? what are the properties of background do you know?

background properties are used to add background effects for elements.
background-color: Sets a solid background color.
background-image: Applies an image as the background.
background-repeat: Controls how the background image repeats (or doesn't).
background-attachment: Controls whether the background is fixed or scrolls with the page.
background-position: Controls the initial position of the background image.
background (shorthand property)

What is overflowing in css?

The overflow property specifies whether to clip the content or to add scrollbars when the content of an element is too big to fit in the specified area.

What is a flex box in css?

CSS Flexbox is a one-dimensional layout model that allows you to design complex web layouts more efficiently and intuitively.

What is a grid layout system?

A grid layout is a multi-dimensional design approach where content is organized into rows and columns

d/f b/w the flex vs grid?

Flexbox: Ideal for simpler, one-dimensional layouts where you need flexibility along a single axis (row or column).
Grid: Best for two-dimensional layouts where you need to control both rows and columns simultaneously and create more complex structures.

What are media queries in css? why do we need to use them?

Media Queries: Allow you to apply CSS styles conditionally, based on characteristics of the device (like width, height, resolution, or orientation).
Need for Media Queries: They enable responsive designs, improving user experience across a variety of devices, making web pages more adaptable, accessible, and user-friendly.

What are transitions in css?

CSS transitions allows you to change property values smoothly, over a given duration.

What are the transforms in css?

CSS transforms allow you to move, rotate, scale, and skew elements.

How can you center a div?

what is d.f b/w the transitions vs transforms vs animations?

Use Transitions: When you want to create smooth changes for simple interactions, like hover effects.
Use Transforms: When you want to change the shape or position of an element without needing to animate it.
Use Animations: When you need to create intricate animations that involve multiple steps or properties over time.

What is a matrix in transforms?

Matrix in Transforms: A compact way to apply multiple transformations (scale, rotate, skew, translate) at once.

d/f b/w css and css3?
